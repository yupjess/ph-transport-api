# PH Transport API

REST API for Philippine fuel prices, LTFRB fares, and vehicle cost estimation.

## Endpoints

- `GET /api/transport/fuel/latest` - Latest fuel prices
- `GET /api/transport/fare/commuter/calculate?type=Modern Jeepney&km=5` - Calculate fare
- `GET /api/transport/health` - Service health
- `GET /api/transport/privacy/policy` - Compliance policy

## Deployment

This project is deployed to Vercel.
